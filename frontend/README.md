# Predicto Frontend

React + Vite + TypeScript + Tailwind CSS.

## Setup

```bash
npm install
```

## Dev

```bash
npm run dev
```

Opens at http://localhost:5173. Expects the backend at http://localhost:8080.

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Stack

- React 19, React Router 7
- @tanstack/react-query
- Axios (with auto Bearer token interceptor)
- Tailwind CSS 3
