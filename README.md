# Predicto 🎮

Komunitná platforma pre esportové predikcie — League of Legends, CS2 a F1.

## Tech Stack
- **Backend:** Spring Boot 3, PostgreSQL, Java 21
- **Frontend:** React 19, TypeScript, Vite
- **Infrastructure:** Docker Compose / Railway

## Quick Start

1. Clone the repo
2. Copy `.env.example` to `.env` and fill in values
3. Run `docker compose up -d`
4. Open `http://localhost:5173`

## Features
- 🎮 Esport predictions (LoL, CS2)
- 🏎️ F1 race predictions
- 🏆 Leaderboard & ELO system
- 🔐 Discord OAuth
- 🌍 Slovak & English support
- ⚡ Auto-sync from PandaScore API

## Deployment
Deployed on Railway. See `.env.example` for required environment variables.
